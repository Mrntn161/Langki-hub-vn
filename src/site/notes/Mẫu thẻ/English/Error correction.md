## Error correction

### Style

#### Material Design Blue Theme

![](https://i.imgur.com/DzpAuIg.png)

![](https://i.imgur.com/K3YbBUP.png)


```css
.card {
    font-family: arial;
    font-size: 14x;
    text-align: center;
    color: black;
    background-color: white;
}

.flashcard-front {
	box-shadow: none;
	border: none;
}


.flashcard-back {
	box-shadow: none;
	border: none;
}


/* Material Design Blue Theme */

.container {
    background: #FFFFFF; /* White Container Background */

    border-radius: 8px;

    font-size: 1.5rem;
    color: #1976D2; /* Material Blue 700 */

    margin-bottom: 15px;
}

.text-container {
    background: #E3F2FD; /* Light Blue Background */
    padding: 20px;
    border-left: 5px solid #1976D2; /* Material Blue 700 */
    border-radius: 8px;
    box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.2);
    margin-bottom: 20px; /* Added spacing */
}

.correct-sentence {
    background: #BBDEFB; /* Slightly darker blue */

    margin-bottom: 20px; /* Added spacing */
}

.explanation {
    background: #E1F5FE; /* Light Cyan Blue */

    box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.2);
    font-style: italic;
    margin-bottom: 20px; /* Added spacing */
}
```
